◆ MODBUS_USE_STATIC_REG_POOL

#define MODBUS_USE_STATIC_REG_POOL   0

Optional memory model: use static pools instead of malloc-backed linked nodes.

Set to 1 for long-running deployments where heap fragmentation is a concern. Memory usage when enabled: (MODBUS_MAX_DIG_REGS * 6) + (MODBUS_MAX_ANA_REGS * 6) bytes. With defaults (64 + 64), this is approximately 768 bytes of static RAM.

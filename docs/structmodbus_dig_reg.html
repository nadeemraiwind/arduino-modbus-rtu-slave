Detailed Description

Linked-node representation for digital register domains.

Memory layout: 6 bytes on AVR (2+1+2+padding), may differ on 32-bit platforms. Structure is naturally aligned for performance. For extreme RAM optimization on ARM/ESP32, consider attribute((packed)), but this typically degrades performance.
